# ionic-livro-app

Projeto de exemplo da disciplina de Programação para Plataformas Móveis

## Execução

Rode o comando abaixo para a instalação das dependências
```
npm install
```

Execução do backend que vai servir os dados para o aplicativo 
```
npm run server
```
